# 5-OS Framework Skill

Instagram'da viral olan "20 tools and 5 folders with 1 system" yaklaşımının KEŞİF versiyonu. Tek kişilik AI-native ürün geliştirme için iskelet.

## Klasör Yapısı

```
D:\0\projeler\<proje-adı>\
├── 01-build-os\      # Ürün inşası (kod, MVP, prototip)
├── 02-think-os\      # Strateji, karar kayıtları, connecting dots
├── 03-make-os\       # İçerik üretimi (volume)
├── 04-ship-os\       # Deployment + uptime + izleme
└── 05-grow-os\       # Pazarlama, monetizasyon, analytics
```

## Her OS'un Araç Seti (20 Araç Toplamda)

### 01-Build OS (Ürün)
- **Claude Code** + `frontend-design` skill
- **GitHub** MCP (versiyon kontrol)
- **Stitch** MCP (UI jenerasyon)
- **21st-dev** MCP (komponent kütüphanesi)

### 02-Think OS (Beyin)
- **Obsidian-vault** MCP (external brain)
- **hybrid-memory** skill (Vector+Graph+Relational)
- **sequential-thinking** MCP
- **notion** MCP (PRD/decision log)

### 03-Make OS (İçerik)
- **ElevenLabs** (ses)
- **Seedance/Higgsfield** (video)
- **Nano Banana 2** (görsel)
- **VibeVoice** (diyalog/podcast)

### 04-Ship OS (Deployment)
- **Cloud Run** MCP
- **Firebase** MCP
- **Supabase** MCP
- **n8n** MCP (otomasyon)

### 05-Grow OS (Büyüme)
- **Crosspost** MCP (sosyal medya)
- **Instagram Manager** skill
- **Ahrefs** MCP (SEO)
- **Stripe/PayPal** (monetizasyon)
